# Konvensi Penamaan Folder Galeri (Tim Media ASIB)

> Panduan lengkap cara memberi nama folder di Google Drive untuk setiap event dokumentasi.

---

## Format Standar Folder

```
{Kategori} — {Nama Event} — {Tanggal}
```

**Contoh:**
```
Keagamaan — Idul Adha 1447 H — 27 Mei 2026
Sosial — Santunan Anak Yatim — 15 Maret 2026
Pendidikan — Wisuda Tahfiz Angkatan 3 — 10 Februari 2026
Umum — Peringatan Hari Gizi — 25 Januari 2026
```

---

## Komponen Penamaan

### 1. Kategori (Wajib)

Pilih salah satu dari 4 kategori:

| Kategori | Icon | Contoh Event |
|----------|------|--------------|
| **Keagamaan** | 🕌 | Idul Adha, Idul Fitri, Kajian Rutin, Tahfiz |
| **Sosial** | 🤝 | Santunan, Baksos, Distribusi Sembako |
| **Pendidikan** | 📚 | Wisuda, Ujian, Kegiatan Belajar |
| **Umum** | 📋 | Rapat, Peringatan Hari Nasional, Workshop |

**Aturan:**
- Awali dengan huruf kapital
- Tidak ada singkatan
- Pisahkan dengan tanda `—` (em-dash)

### 2. Nama Event (Wajib)

**Format:** Deskriptif, jelas, dengan detail penting

**Contoh yang BENAR:**
```
Idul Adha 1447 H
Santunan Anak Yatim Ramadhan
Wisuda Tahfiz Angkatan 3
Kajian Pekanan Tafsir Juz 30
Peringatan Hari Gizi Nasional 2026
```

**Contoh yang SALAH:**
```
idul adha               ❌ Huruf kecil semua
IDUL ADHA               ❌ Huruf kapital semua
IdulAdha                ❌ Tanpa spasi
Event Hari Ini          ❌ Tidak deskriptif
Galeri Foto             ❌ Generik
```

**Tips Nama Event:**
- Sertakan tahun Hijriyah untuk event keagamaan: `Idul Adha 1447 H`
- Sertakan angka/batch untuk event berulang: `Angkatan 3`, `Batch 2`
- Gunakan bahasa Indonesia yang baik
- Hindari singkatan yang tidak umum

### 3. Tanggal (Wajib)

**Format:** `{Hari} {Bulan} {Tahun}`

**Contoh:**
```
27 Mei 2026
15 Maret 2026
10 Februari 2026
1 Januari 2026
```

**Aturan:**
- Tanggal tanpa leading zero (tulis `5` bukan `05`)
- Nama bulan lengkap dalam bahasa Indonesia
- Tahun 4 digit
- Pisahkan dengan tanda `—` (em-dash)

## Workflow Membuat Folder Baru

### Step 1: Buka Google Drive

1. Buka https://drive.google.com di browser
2. Login dengan akun tim media: timitasib@gmail.com

### Step 2: Buat Folder Baru

**Cara A: Tombol New**
1. Klik tombol **"+ New"** (kiri atas, berwarna warna-warni)
2. Pilih **"New folder"**
3. Muncul popup "Untitled folder"

**Cara B: Klik Kanan**
1. Klik kanan di area kosong Google Drive
2. Pilih **"New folder"**
3. Muncul folder dengan nama default

### Step 3: Tulis Nama Folder

1. Hapus nama default ("Untitled folder")
2. Ketik nama sesuai format:
   ```
   Keagamaan — Idul Adha 1447 H — 27 Mei 2026
   ```
3. Tekan **Enter**

**Tips mengetik tanda em-dash (`—`):**
- Windows: `Alt + 0151` (tahan Alt, ketik 0151 di numpad)
- Mac: `Option + Shift + -` (tanda minus)
- Atau copy-paste dari contoh ini: `—`
- Alternatif: gunakan tanda minus biasa `-` jika sulit

### Step 4: Share ke Service Account

1. Klik kanan folder yang baru dibuat
2. Pilih **"Share"**
3. Di kolom "Add people", paste email:
   ```
   asib-gallery-reader@gen-lang-client-0412959743.iam.gserviceaccount.com
   ```
4. Klik dropdown permission → pilih **"Viewer"**
5. Klik **"Share"**

### Step 5: Verifikasi

1. Folder sekarang muncul di daftar
2. Icon folder berwarna putih (bukan warna-warni)
3. Ada icon "person" kecil di kanan bawah folder (tanda sudah di-share)
4. Buka folder → kosong (siap diisi foto)

## FAQ Folder Naming

### Q: Apakah bisa pakai tanda minus (`-`) bukan em-dash (`—`)?

**A:** Bisa! Gunakan yang lebih mudah:
```
Keagamaan - Idul Adha 1447 H - 27 Mei 2026
```

### Q: Bagaimana jika event跨 beberapa hari?

**A:** Gunakan tanggal pertama atau tanggal utama:
```
Keagamaan — Idul Adha 1447 H — 27-29 Mei 2026
atau
Keagamaan — Idul Adha 1447 H — 27 Mei 2026
```

### Q: Apakah bisa rename folder setelah dibuat?

**A:** Bisa! Klik kanan folder → Rename. Folder ID tidak berubah, jadi tidak masalah untuk website.

### Q: Bagaimana jika tanggal belum pasti (rencana)?

**A:** Gunakan estimasi atau tulis "TBD":
```
Sosial — Santunan Rencana — TBD 2026
```
Setelah tanggal fix, rename folder.

### Q: Bisa pakai bahasa Inggris?

**A:** Tidak direkomendasikan. Gunakan bahasa Indonesia untuk konsistensi:
```
❌ Religious — Eid al-Adha 1447 H — 27 May 2026
✅ Keagamaan — Idul Adha 1447 H — 27 Mei 2026
```

### Q: Bagaimana dengan jam di folder name?

**A:** Tidak perlu. Jam di folder name tidak perlu karena sudah ada di metadata foto (timestamp kamera). Tanggal saja cukup.
