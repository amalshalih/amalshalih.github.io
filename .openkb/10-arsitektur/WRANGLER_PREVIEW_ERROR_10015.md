# 🚨 IMPORTANT: Worker Previews Error 10015 - BUKAN Bug!

## Problem

```bash
$ wrangler preview
✘ [ERROR] A request to the Cloudflare API failed.
  
  You do not have access to use Worker Previews. [code: 10015]
```

## ✅ SOLUSI: Gunakan `wrangler dev` BUKAN `wrangler preview`

### Perbedaan:

| Command | Fungsi | Status | Butuh Izin? |
|---------|--------|--------|-------------|
| `wrangler dev` | Local development server | **✅ GA** (General Availability) | Tidak |
| `wrangler preview` | Preview deployment URL | 🔒 **Private Beta** | Ya (Enterprise/beta) |

---

## 🔧 Workflow yang Benar (Astro v6 + Cloudflare Workers)

### 1. Development Lokal

```bash
# Build dulu
bun run build

# Jalankan wrangler dev dari dist/server (tempat wrangler.json auto-generated)
cd dist/server && npx wrangler dev

# Atau pakai script shortcut
bun run wrangler:dev
```

### 2. Deploy Production

```bash
# Build
bun run build

# Deploy dari dist/server
cd dist/server && npx wrangler deploy

# Atau pakai script shortcut  
bun run wrangler:deploy
```
